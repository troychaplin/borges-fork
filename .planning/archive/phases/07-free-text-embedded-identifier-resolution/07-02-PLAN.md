identifiers to the existing DOI/PMID resolver backends, degrade gracefully to the
freetext parser and then SUPPORTED_INPUT_MESSAGE on failure, and confirm the

DOI/PMID now resolves instead of failing through the brittle heuristic parser, with
zero new SSRF surface and no regression to bare-identifier handling.

covered by a new integration describe block using the existing makeFetchFn /
citation-js-mocks patterns.
